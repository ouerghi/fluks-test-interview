

# getQuestion

## Install

Use `composer`

- https://getcomposer.org/download/

```
$ composer install
```


## Usage

Count number of questions about *pompe* topic:

```
$ php getQuestion.php count pompe
```

result

```
count for 'pompe' = 300
```

## Unit tests

To run unit tests

```
$ ./vendor/bin/phpunit QuestionTest.php
```
